CLI Corporativo

Use o CLI do Claude Code em ambientes corporativos com proxies, SSO, políticas de segurança e restrições de rede.

// Configuração de proxy corporativo
export ANTHROPIC_API_URL="https://proxy.empresa.com"
export ANTHROPIC_API_KEY="sk-ant-..."
// Configuração de certificados SSL
export NODE_EXTRA_CA_CERTS="/path/to/corp-ca.crt"
$ claude --no-update-check
