Seleção de Modelos

Escolha o Model certo para cada tarefa. Compare Claude Opus, Sonnet e Haiku: custo, velocidade, capacidade e casos de uso ideais.

// Selecionar modelo via flag
$ claude --model claude-opus-4-5 "tarefa complexa"
$ claude --model claude-haiku-3-5 "tarefa simples"
// Configurar modelo padrão em settings.json
{
  "model": "claude-sonnet-4-5",
  "smallModel": "claude-haiku-3-5"
}
