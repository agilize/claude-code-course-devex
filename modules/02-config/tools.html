Ferramentas

O Claude Code dispõe de um conjunto poderoso de Tools built-in: Read, Write, Bash, Glob, Grep e mais. Aprenda a configurar e usar cada ferramenta.

// Ferramentas disponíveis
Read, Write, Edit, Bash, Glob, Grep
WebFetch, WebSearch, TodoWrite
// Permitir apenas ferramentas específicas
$ claude --allowedTools "Read,Write,Bash"
