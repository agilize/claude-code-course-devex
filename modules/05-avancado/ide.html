IDE Integration

Integre o Claude Code com seu IDE favorito: VS Code, JetBrains, Vim/Neovim. Aprenda a configurar extensões e fluxos de trabalho híbridos.

// VS Code — abrir terminal integrado
Ctrl+` → digitar: claude

// Extensão Claude Code para VS Code
ext install anthropic.claude-code
// JetBrains — plugin oficial
Settings → Plugins → Claude Code
# Disponível para IntelliJ, WebStorm, PyCharm
