CLI Completo

Domine a interface de linha de comando do CLI do Claude Code. Aprenda todos os flags, comandos e opções disponíveis.

$ claude --print "gere um README"
$ claude --output-format json
$ claude --max-turns 10
$ claude --allowedTools "Read,Write,Bash"
$ claude --disableAutoUpdate
