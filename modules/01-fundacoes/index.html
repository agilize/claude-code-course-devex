O que é Claude Code

Claude Code é a CLI oficial da Anthropic para desenvolvimento assistido por IA. Neste módulo você aprende os fundamentos do Claude Code e como ele transforma seu fluxo de trabalho.

$ claude --help
$ claude "explique este código"
$ claude --version
$ npm install -g @anthropic-ai/claude-code
