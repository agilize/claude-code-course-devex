Sistema de Memória

Entenda como o Claude Code gerencia a Memória entre sessões: arquivos CLAUDE.md, memória de projeto e memória global.

// Níveis de memória
~/.claude/CLAUDE.md          # memória global
.claude/CLAUDE.md            # memória de projeto
subdir/CLAUDE.md             # memória de subdiretório
// Comando para editar memória
/memory
# Abre o editor de memória interativo
