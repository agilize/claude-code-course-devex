Referência rápida de comandos

Os comandos mais importantes do Claude Code para consulta rápida.

# Instalação e setup
npm install -g @anthropic-ai/claude-code
claude login
claude --version

# Sessão interativa
claude                        # inicia sessão
claude "sua pergunta"         # pergunta direta
claude --model claude-opus-4-6 # escolhe modelo

# Flags essenciais
claude --verbose              # mostra chamadas de ferramentas
claude --no-auto-accept       # review manual de cada ação
claude --output-format json   # saída em JSON
claude --dangerously-skip-permissions  # skip confirmações (cuidado)

# Comandos de sessão (dentro da sessão)
/status                       # tokens, modelo, custo
/compact                      # compacta o contexto
/model claude-haiku-4-5       # troca modelo
/clear                        # limpa o contexto
/help                         # lista comandos disponíveis

# Passando input
cat arquivo.ts | claude --stdin "explique este código"
git diff | claude --stdin "revise este diff"

Estrutura de arquivos de configuração

Todos os arquivos de configuração relevantes do Claude Code e onde encontrá-los.

# Configurações globais (afetam todas as sessões)
~/.claude/settings.json       # configurações pessoais globais
~/.claude/CLAUDE.md           # instruções globais (para todos os projetos)
~/.claude/keybindings.json    # atalhos de teclado personalizados

# Configurações do projeto (versionadas no repositório)
.claude/settings.json         # configurações do projeto
CLAUDE.md                     # instruções do projeto
.claudeignore                 # arquivos ignorados pelo Claude

# Histórico e sessões
~/.claude/sessions/           # histórico de sessões anteriores
~/.claude/logs/               # logs de atividade

# Extensões IDE
VS Code:     ~/.vscode/extensions/anthropic.claude-code-*
JetBrains:   ~/.config/JetBrains/*/plugins/claude-code/

Exercício Final

Parabéns por concluir o curso. Para consolidar o aprendizado e contribuir com a comunidade:

  1. Configure seu ambiente definitivo: Crie ou atualize o CLAUDE.md do seu projeto principal com tudo que aprendeu. Inclua stack, comandos, convenções, arquivos críticos e padrões.
  2. Explore um MCP server: Instale o MCP server do GitHub ou filesystem e configure-o no seu settings.json. Teste pelo menos uma operação via Claude.
  3. Compartilhe com sua equipe: Apresente 3 técnicas do curso para um colega de trabalho — CLAUDE.md, hooks ou agent teams. Documente os prompts que funcionaram melhor no seu contexto.
  4. Contribua: Se encontrar um bug no Claude Code ou tiver uma sugestão, abra uma issue no repositório oficial.
